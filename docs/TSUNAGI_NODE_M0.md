# TSUNAGI Node M0

## 日本語

M0は「検証できるリレー級ノード」の最小定義です。  
同期と検証ができる、現場で安心して使える基盤を目指します。  
TSUNAGI NodeはZigのみでゼロから作ります。  
方針の詳細: `docs/ZIG_ONLY_POLICY.md`  

非目標:
- ブロック生成（BP）は含みません
- 秘密鍵の運用は含みません

## English

M0 is the minimum definition of a relay‑class validating node.  
It aims to be a safe, practical base for sync and validation.  
TSUNAGI Node will be built Zig‑only from the ground up.  
Policy details: `docs/ZIG_ONLY_POLICY.md`  

Non‑goals:
- No block production (BP)
- No key management
